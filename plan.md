# Doctors Booking Appointment System (Front-End Only)

## Notes
- System will be built entirely with HTML, CSS, and JavaScript.
- All data (users, appointments) will be stored in browser localStorage as JSON arrays.
- No backend or server; app runs directly in the browser.
- Three user roles: Patient, Doctor, Admin, with role-based dashboards and permissions.
- Modular JS files: auth.js, storage.js, ui.js, etc.
- Sample data will be preloaded on first run.
